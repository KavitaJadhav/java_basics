# Java Basics 🖥️

A structured repository covering **core Java concepts**, syntax, and examples, designed for learning, quick revision, and building a solid foundation in Java programming.

---

## 📌 Summary

This repository contains examples and explanations of **fundamental Java concepts**, including object-oriented programming, data types, collections, exception handling, and more. It is suitable for beginners, as well as for quick refresher exercises for experienced developers.

---

## 🧠 Key Topics

* Java Syntax & Data Types
* Variables & Operators
* Control Flow (if-else, loops)
* Methods & Functions
* Object-Oriented Programming (Classes, Objects, Inheritance, Polymorphism, Encapsulation, Abstraction)
* Exception Handling
* Collections Framework (List, Set, Map)
* Generics
* File I/O & Streams

## 🚀 Example

### ArrayList Example

```java id="javaex1"
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        ArrayList<String> fruits = new ArrayList<>();
        fruits.add("Apple");
        fruits.add("Banana");
        System.out.println(fruits);
    }
}
```
